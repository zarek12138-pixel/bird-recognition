# 鸟类叫声识别网站

一个精美的鸟类叫声识别网站，支持上传音频进行AI识别。

## 功能特性

- 🎤 上传音频文件识别鸟类
- 📚 鸟类图鉴浏览（27种鸟类）
- 📊 识别历史记录
- 🎨 精美的动态UI设计
- 📱 响应式设计，支持手机访问

## 在线访问

部署到GitHub Pages后，访问地址：
```
https://你的用户名.github.io/仓库名/
```

## 技术栈

- HTML5 + CSS3 + JavaScript
- Font Awesome 图标
- Google Fonts 字体

## 部署说明

1. 将此文件夹内容上传到GitHub仓库
2. 进入 Settings → Pages
3. Source 选择 Deploy from a branch
4. Branch 选择 main 和 / (root)
5. 点击 Save，等待1-2分钟即可访问

## 本地预览

直接双击 `index.html` 文件即可在浏览器中预览。
